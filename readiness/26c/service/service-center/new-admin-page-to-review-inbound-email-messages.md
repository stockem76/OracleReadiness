# New Admin Page to Review Inbound Email Messages

`⚙️ Steps to Enable`

> 📖 [View on Oracle Help Center](https://docs.oracle.com/en/cloud/saas/readiness/service/26c/bsvc-26c/26C-b2b-service-wn-f43170.htm)

The new Email Monitoring administrative page enables administrators to view the processing status of inbound service emails. An administrator can view a list of the most recent email messages and optionally can search by a particular message ID.  The table on the page provides details about each message such as the email channel used, date, status, and other metadata. If a message experienced a processing failure, the administrator can use the inline action to reprocess the email, for example after they have taken some corrective action to resolve the failure reason.

To access the page, navigate to Service Center Administration, then search for Email Monitoring.

This new page gives the administrator more visibility into email process status and reduces the need to contact Oracle Support for assistance in reprocessing messages.

## ⚙️ Steps to Enable and Configure

This administrative feature is enabled automatically.
