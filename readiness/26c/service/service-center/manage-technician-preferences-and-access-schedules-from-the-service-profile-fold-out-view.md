# Manage technician preferences and access schedules from the Service Profile fold-out view

`⚙️ Steps to Enable`

> 📖 [View on Oracle Help Center](https://docs.oracle.com/en/cloud/saas/readiness/service/26c/bsvc-26c/26C-b2b-service-wn-f46525.htm)

You can now use the service profile fold-out page to manage technician preferences and access schedules for a given site / service profile. Previously this required navigation to the details page but with this enhancement, we've introduced a new fold-out panel for Technician Preferences, as well as smart actions and corresponding sub-views for both Technician Preferences and Access Hours.

Users can see all items related to a service profile from the fold-out view and it's sub-views, avoiding extra clicks to navigate, and providing improved capabilities like searching and sorting in a more intuitive user experience.

## ⚙️ Steps to Enable and Configure

You don't need to do anything to enable this feature.
