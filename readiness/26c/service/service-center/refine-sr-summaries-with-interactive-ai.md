# Refine SR Summaries with interactive AI

`⚙️ Steps to Enable` · `💡 Tips & Considerations` · `🔐 Access Requirements`

> 📖 [View on Oracle Help Center](https://docs.oracle.com/en/cloud/saas/readiness/service/26c/bsvc-26c/26C-b2b-service-wn-f46427.htm)

Service representatives can now interact directly with the SR Summary.

• Click the SR Summary section to activate it
• Enter prompt-based commands using the action bar (for example: “simplify”, “add more details”, “focus on recent updates”)
• The system regenerates the summary in real time based on the input

The updated summary appears instantly without reloading the page.

• Helps improve clarity of AI-generated summaries for the user
• Helps agents quickly understand SR context
• Builds trust in AI by allowing refinement of the output

## ⚙️ Steps to Enable and Configure

Refine SR Summaries with Interactive AI

1. Navigate to the Service Request (SR) details page
2. Locate the SR Summary section
3. Click on the SR Summary to activate interactive mode
4. Use the action bar to enter a prompt command
5. Submit the prompt
6. View the refreshed summary generated by AI in real time
7. Repeat prompts as needed to further refine the summary

*(Note: Ensure AI Summarization and related AI Agent services are enabled and properly configured in your environment.)*
