# Initiate Smart Actions added to the SR action plan directly from the SR Pending Section

`⚙️ Steps to Enable`

> 📖 [View on Oracle Help Center](https://docs.oracle.com/en/cloud/saas/readiness/service/26c/bsvc-26c/26C-b2b-service-wn-f47321.htm)

Customer service representatives can streamline service request handling by accessing Smart Actions directly from the Pending section of an SR.

When an action is added from the action bar (using "Add Action..."), the action will show in the Pending section. Supported actions can be edited inline, enabling users to quickly open the action, enter required details, and complete the action without leaving the Pending section.

The following Smart Actions are supported:

• Resolve SR
• Update Related Objects
• Log a Call
• Start Internal Conversation
• Create Work Order
• Create Relationship
• Compose Web Message
• Compose Message
• Compose Internal Note
• Compose Email
• Add Attachment
• Transfer SR

Create Work Order, Relationship, Compose Web Message or Message from the Pending Section of an SR

Simplifies the UI and reduces clicks by surfacing relevant next steps directly within pending service requests.

## ⚙️ Steps to Enable and Configure

You don't need to do anything to enable this feature.
