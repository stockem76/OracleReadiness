# Transaction Type/Declaration Type Available on Line UIs

`⚙️ Steps to Enable`

> 📖 [View on Oracle Help Center](https://docs.oracle.com/en/cloud/saas/readiness/logistics/26c/gtm26c/26C-gtm-wn-f49602.htm)

The Type ID is now available in the Trade Transaction Line and Declaration Line user interfaces.

• For Trade Transaction Lines, the Transaction Type is available in the finder, finder results, view, and edit pages.
• For Declaration Lines, the Declaration Type is available in the finder, finder results, view, and edit pages.

Transaction Line - Transaction Type ID

**Business Benefit:**This enhancement improves visibility and access to transaction type and declaration type information directly within key workflow pages. Users can more efficiently search, review, and manage records without navigating to additional screens.

## ⚙️ Steps to Enable and Configure

You don't need to do anything to enable this feature.
