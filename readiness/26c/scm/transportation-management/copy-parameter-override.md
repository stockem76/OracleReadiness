# Copy Parameter Override

`⚙️ Steps to Enable`

> 📖 [View on Oracle Help Center](https://docs.oracle.com/en/cloud/saas/readiness/logistics/26c/otm26c/26C-otm-wn-f46332.htm)

This feature provides you with the option of copying your Parameter Override.

The new Copy Parameter Override action can be found by navigating to Logistics Network Modeling > Power Data > Parameter Overrides.

Copy Parameter Override Action

When the Copy Parameter Override action is select the Override Parameters from the selected Override Parameter set are copied.  The copied Override Parameter can be modified and assigned to a new ID.

Parameter Override Copy Example

When finished, in this example, the original and newly copied Parameter Override are now both available for modeling usage.

Original and Copied Parameter Override

**Business Benefit:**This usability enhancement allows you to duplicate an existing, working Parameter Override and then adjust only what’s different. This improves productivity and consistency across scenarios, reduces manual re-entry and misconfiguration risk.

## ⚙️ Steps to Enable and Configure

You don't need to do anything to enable this feature.
