# Fetch All Shipment Tracking Events

`⚙️ Steps to Enable`

> 📖 [View on Oracle Help Center](https://docs.oracle.com/en/cloud/saas/readiness/logistics/26c/otm26c/26C-otm-wn-f47436.htm)

When this Optional Feature is enabled, the application retrieves tracking events matched via LEGACY SHIPMENT MATCHING and copied via COPY SHIPMENT EVENTS, honoring the Hidden checkbox and the Display Hidden Shipment Events user preference. When disabled, only tracking events associated with a shipment ID are displayed.

**Business Benefit:** This feature gives you a more complete view of your shipment's tracking event activity.

## ⚙️ Steps to Enable and Configure

If you need to change the Opt In state for this feature:

• Go to the Optional Feature UI - **Configuration and Administration > Property Management > Optional Features**.

Your user must have the DBA.ADMIN user role to use this functionality.

• Select the**Fetch All Shipment Tracking Events**feature.
• Run the desired Action for the feature - Opt In or Opt Out.
