# Use 3D Load Config for Ground Schedule and Consol

`⚙️ Steps to Enable`

> 📖 [View on Oracle Help Center](https://docs.oracle.com/en/cloud/saas/readiness/logistics/26c/otm26c/26C-otm-wn-f47442.htm)

This Optional Feature, when enabled, will use your 3D load configuration for actions involving Ground Schedule and consol (GROUND or OCEAN FCL) shipments.

**Business Benefit:** For Grounds Schedule and Consols (Ground and FTL), this feature improves solution quality by optimizing placement of both existing and newly added freight.

## ⚙️ Steps to Enable and Configure

If you need to change the Opt In state for this feature:

• Go to the Optional Feature UI - **Configuration and Administration > Property Management > Optional Features**.

Your user must have the DBA.ADMIN user role to use this functionality.

• Select the**Use 3D Load Config for Ground Schedule and Consol**feature.
• Run the desired Action for the feature - Opt In or Opt Out.
