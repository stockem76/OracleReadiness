# Selected Shipping Bug Fixes in This Update

`⚙️ Steps to Enable`

> 📖 [View on Oracle Help Center](https://docs.oracle.com/en/cloud/saas/readiness/scm/26c/inv26c/26C-inventory-wn-f49922.htm)

This update includes some bug fixes that can change the way Oracle Shipping works. This isn't a full list of all the bug fixes in this update. This list includes the bug fixes that can cause a noticeable change in application behavior.

**Delete Shipping Transaction Requests Using Download Shipping Transaction Correction Records in Spreadsheet**

With this update, you can delete shipping transaction requests using the Download Shipping Transaction Correction Records in Spreadsheet feature.

To delete records:

1. Select **Download Data** to search for interface transactions.
2. After the records are downloaded, select the rows to delete.
3. Choose **Table Row Changes**, and then select **Mark for Action** from the drop-down list.
4. Select **Delete Shipment Request**.
5. Select **Upload Changes** to delete the records marked for deletion.

Oracle reference: 38931089

## ⚙️ Steps to Enable and Configure

You don't need to do anything to enable this feature.
