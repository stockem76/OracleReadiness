# AI Agent - AI Workflow: Inventory Organization Update Assistant

`⚙️ Steps to Enable`

> 📖 [View on Oracle Help Center](https://docs.oracle.com/en/cloud/saas/readiness/scm/26c/inv26c/26C-inventory-wn-f49393.htm)

Use the Inventory Organization Update Assistant to apply mass updates and modify end-user access for selected organizations. You can filter your selection using a wide range of criteria. You can also upload a spreadsheet to create organizations.

The organization assistant has been refactored to use a workflow agent, and enhanced to support bulk data creation using a CSV formatted file to specify organization details.

## ⚙️ Steps to Enable and Configure

You don't need to do anything to enable this feature.
