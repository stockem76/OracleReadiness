# Continued Redwood Migration for Web UI

`⚙️ Steps to Enable`

> 📖 [View on Oracle Help Center](https://docs.oracle.com/en/cloud/saas/readiness/logistics/26c/wms26c/26C-wms-wn-f49713.htm)

Additional screens within WMS Redwood Desktop have been redesigned with the Redwood experience, bringing continued usability and productivity enhancements. These updated screens are easily accessible from the Home page through **Ask Oracle**.

The redesigned global and in-application navigation bars make it easier to move across system functions, while enhanced dashboards provide quick visibility into operational performance and health. Users can also personalize flexible page layouts to better align with their individual workflow preferences.

## ⚙️ Steps to Enable and Configure

To access the newly available redwood screens:

1. From the **User** drop-down, select -> **Try the new Redwood Experience.**
2. After the Redwood screen opens, click the **Home with Ask Oracle** button in the bottom right.
3. From the **Home with** **Ask Oracle page,**you can choose a category and then find all screens available underneath each selected category.
