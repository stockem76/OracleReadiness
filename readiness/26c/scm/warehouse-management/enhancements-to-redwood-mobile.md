# Enhancements to Redwood Mobile

`⚙️ Steps to Enable`

> 📖 [View on Oracle Help Center](https://docs.oracle.com/en/cloud/saas/readiness/logistics/26c/wms26c/26C-wms-wn-f50399.htm)

**Increased Resiliency with Barcode Scanners**

When Redwood mobile is used with barcode scanners (integrated scanners, USB scanners etc.), the scanning mechanism has been improved to be more resilient and forgiving than before, resulting in fewer missed or partial scans, thereby improving overall scanning efficiency. As part of this improvement, the visual indicator that RW Mobile uses during scan processing will change from a spinning circle to a less intrusive … indicator next to the page title, along with other subtle changes that improve the transitions between scans.

**Display Density Settings**

Redwood mobile also now has added flexibility to handle different screen sizes more efficiently by allowing the user to adjust the scale factor between Small/Medium/Large. In earlier releases the default was Large. The default is now “Medium” which utilizes the space more efficiently across more screen sizes. For even smaller screens, users can try “Small”, while “Large" is better for larger mobile devices or tablets. The scale factor can be set at the Group level and can be overridden by users via the Mobile user profile page. User level overrides only persist on the specific device where the change was done and for that user only.

## ⚙️ Steps to Enable and Configure

You can set the scale factor for Redwood Mobile screens at the Group level. These settings can be overridden by users via the Mobile **user profile** page. User level overrides only persist on the specific device where the change was done and for that user only.
