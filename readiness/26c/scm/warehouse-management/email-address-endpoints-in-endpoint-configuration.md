# Email Address Endpoints in Endpoint Configuration

`⚙️ Steps to Enable`

> 📖 [View on Oracle Help Center](https://docs.oracle.com/en/cloud/saas/readiness/logistics/26c/wms26c/26C-wms-wn-f50177.htm)

Endpoint Configuration now supports email address endpoints for output interfaces. Users can now select or configure an endpoint type of Email Address.

Users can enter a valid email address as the endpoint destination.

• WMS validates the email address before saving.
• Existing output interface configurations that use email endpoints can be viewed and edited. Invalid email addresses display a validation message and cannot be saved.

**NOTE:** Existing non-email endpoint behavior remains unchanged.

## ⚙️ Steps to Enable and Configure

1. Open Endpoint Configuration. Select Email Address as the endpoint type.
2. Enter a valid email address.
3. Save the endpoint.
4. Update or create output interface configurations that need email-based destinations.
