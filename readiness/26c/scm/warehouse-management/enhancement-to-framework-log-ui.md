# Enhancement to Framework Log UI

`⚙️ Steps to Enable`

> 📖 [View on Oracle Help Center](https://docs.oracle.com/en/cloud/saas/readiness/logistics/26c/wms26c/26C-wms-wn-f50176.htm)

We’ve enhanced the Framework Logs UI to include the users who created each log entry. A new column **create_user** has been introduced.

The column is hidden, by default. Users can add the column to the grid and include it in exports. CSV export log descriptions now include the source screen name.

## ⚙️ Steps to Enable and Configure

1. Open Framework Logs.
2. Add the create_user column to the grid if needed.
3. Include the column in exports if users need audit visibility.
