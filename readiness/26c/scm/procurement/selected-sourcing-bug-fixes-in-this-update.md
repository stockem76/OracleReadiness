# Selected Sourcing Bug Fixes in This Update

`⚙️ Steps to Enable`

> 📖 [View on Oracle Help Center](https://docs.oracle.com/en/cloud/saas/readiness/scm/26c/proc26c/26C-procurement-wn-f49711.htm)

This update includes a bug fix that can change the way Oracle Sourcing works. This isn't a full list of all the bug fixes in this update.

**Redwood: Latest Negotiation Style Changes are Applied When Creating Negotiation from Template**

When a negotiation is created in Redwood with a template, it enables features in the negotiation style version used for creating the template. But prior to update 26C, if the style was later updated to enable or disable new features, those are not reflected in the negotiation. This issue is fixed in update 26C. When creating a negotiation from a template, the latest negotiation style version is applied, ensuring any updates to existing features or new features enabled are applied and reflected in the negotiation.

## ⚙️ Steps to Enable and Configure

You don't need to do anything to enable this feature.
