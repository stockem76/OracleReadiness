# Selected External Purchase Prices Bug Fixes in This Update

`⚙️ Steps to Enable`

> 📖 [View on Oracle Help Center](https://docs.oracle.com/en/cloud/saas/readiness/scm/26c/proc26c/26C-procurement-wn-f49435.htm)

This update includes some bug fixes that can change the way Oracle External Purchase Prices works. This isn't a full list of all the bug fixes in this update. This list includes the bug fixes that can cause a noticeable change in application behavior.

**Import External Purchase Prices Using FBDI Templates on Mac Computers**

Before update 26C, you could import External Purchase Prices records using file-based data import (FBDI) templates only on Microsoft Windows computers. After update 26C, you can download and use the latest FBDI templates to import External Purchase Prices records on MAC computers, too.

Oracle reference: 39324010

## ⚙️ Steps to Enable and Configure

You don't need to do anything to enable this feature.
