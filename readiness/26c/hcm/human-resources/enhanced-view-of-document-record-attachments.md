# Enhanced View of Document Record Attachments

`⚙️ Steps to Enable`

> 📖 [View on Oracle Help Center](https://docs.oracle.com/en/cloud/saas/readiness/hcm/26c/hure-26c/26C-hr-wn-f50471.htm)

You can now view attachments associated with a document record directly by selecting the attachment count hyperlink. Clicking the attachment count opens the associated attachments in a side panel, allowing you to review document record attachments without navigating away from the document record page.

This streamlined experience provides faster and more convenient access to supporting documents.

Attachment Count Hyperlink on Redwood Document Records Page

You can preview the attachment directly on the page and download it when needed.

View Attachment Directly on Redwood Document Records

When multiple attachments are available, you can browse them directly in the side panel using the Next and Previous navigation arrows, making it easy to move between attachments without leaving the page.

View All Attachments

Reduces the number of steps needed to open related documents and helps users work more efficiently when reviewing or managing document records.

## ⚙️ Steps to Enable and Configure

You don't need to do anything to enable this feature.

For more information about document records, refer to the Using Global Human Resources guide on Oracle Help Center.
