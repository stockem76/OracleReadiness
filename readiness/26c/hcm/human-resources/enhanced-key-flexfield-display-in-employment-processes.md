# Enhanced Key Flexfield Display in Employment Processes

`⚙️ Steps to Enable`

> 📖 [View on Oracle Help Center](https://docs.oracle.com/en/cloud/saas/readiness/hcm/26c/hure-26c/26C-hr-wn-f47393.htm)

The Employment Info and Employment Details pages now use the standard key flexfield component for People Group and Default Expense Account information. This provides a more consistent Redwood experience for key flexfield values and enables users to view combination details for these attributes.

People Group key flexfield display

Combination details

This enhancement improves usability and consistency for employment information by showing People Group and Default Expense Account values through the standard key flexfield experience. It also helps uptaking pages and fragments use the same REST-backed identifiers needed to render key flexfield details correctly.

## ⚙️ Steps to Enable and Configure

You don't need to do anything to enable this feature.
