# Redwood Experience for Wellness Employee

`⚙️ Steps to Enable`

> 📖 [View on Oracle Help Center](https://docs.oracle.com/en/cloud/saas/readiness/hcm/26c/hure-26c/26C-hr-wn-f50133.htm)

We have enriched the user experience with the new Wellness Application developed using the Redwood toolset. The pages are built from the ground up using Visual Builder Studio (VB Studio) to give you a unique experience of Oracle applications.

You can navigate to the page by using the existing app icon **Wellness**under the **Me**tab on the home page. Wellness is now added as an application under **Corporate** **Citizenship**. So, you can also navigate to Wellness Application via **Corporate Citizenship**under the **Me**tab.

The Redwood Wellness pages are the equivalent of the existing Wellness pages and has the same functionality, including validations.
