# Configure HCM Search Views for the Redwood Jobs Search Page

`⚙️ Steps to Enable` · `🔐 Access Requirements`

> 📖 [View on Oracle Help Center](https://docs.oracle.com/en/cloud/saas/readiness/hcm/26c/hure-26c/26C-hr-wn-f44424.htm)

You can now customize and streamline the Redwood Jobs search experience by configuring HCM Search Views for the Redwood Jobs search page.

For more information, see the **Steps to Enable and Configure** section of this What’s New.

This feature enables you to customize and configure available sets of fields on the Redwood Jobs search page based on your business requirements. This helps you control the keyword search fields, filters, and display fields based on the seeded set of standard fields.

## ⚙️ Steps to Enable and Configure

For more information, see Configure HCM Search Views for the Redwood Jobs Search Page.

## 🔐 Access Requirements

You can configure HCM Search Views for the Redwood Jobs search page if your role has this privilege: **Manage Search Views (HRC_MANAGE_SEARCH_VIEWS)**.
