# Filter Collective Agreement Grades by Selected Collective Agreement

`⚙️ Steps to Enable`

> 📖 [View on Oracle Help Center](https://docs.oracle.com/en/cloud/saas/readiness/hcm/26c/hure-26c/26C-hr-wn-f49632.htm)

The Collective Agreements page now includes a France Collective Agreement Information section where you can associate grades with a collective agreement. In hire and assignment flows, the Collective Agreement Grade field now shows only the grades linked to the collective agreement you selected.

Now you can:

• Add one or more grades to a France collective agreement in the France Collective Agreement Information section.
• See only the grades linked to the selected collective agreement in the Collective Agreement Grade field.
• Show a single grade when only one grade is linked to the collective agreement.
• Show all linked grades when multiple grades are linked to the collective agreement.
• Leave the Collective Agreement Grade field blank when no grade is linked to the selected collective agreement.

Make use of this feature to filter the collective agreement grades.

## ⚙️ Steps to Enable and Configure

• 1. Navigate to task Manage Common Lookups
2. Select lookup ORA_HRX_FR_CAGR_GRADE
3. Add all the values of collective agreement grade applicable to the implementation: so if multiple collective agreements are in place all the values for all te collective agreements should be added.
4. Navigate to task Manage Collective Agreements
5. Select the collective agreement for which you want to define the grades (e.g. Syntec)
6. Navigate to section France Collective Agreement Information
7. Add the applicable collective agreement grades (click on the plus and the pencil icons to add)
