# Enable SMS for Nudges Related to Journey Reminders

`⚙️ Steps to Enable` · `💡 Tips & Considerations` · `🔐 Access Requirements`

> 📖 [View on Oracle Help Center](https://docs.oracle.com/en/cloud/saas/readiness/hcm/26c/hure-26c/26C-hr-wn-f49479.htm)

You can enable SMS notifications for nudges related to Journey reminders. This enhancement extends Journey reminder nudges so that workers can receive reminder communications by SMS.

Configure SMS Channel for Journey Nudge

Nudge Notification Received via SMS

SMS delivery for Journey reminder nudges can help organizations reach workers through an additional communication channel, supporting timely action.

## ⚙️ Steps to Enable and Configure

You don't need to do anything to enable this feature.

## 💡 Tips and Considerations

• SMS notifications are sent only to the work mobile number, so a work mobile phone number must be included in the person record.

## 🔐 Access Requirements

You must be granted the Nudge Configuration (ORA_PER_NUDGE_CONFIGURATION_DUTY) duty role to configure nudges.

## 📚 Key Resources

For more information about nudges, refer to the HCM Nudges topic on Oracle Help Center.
