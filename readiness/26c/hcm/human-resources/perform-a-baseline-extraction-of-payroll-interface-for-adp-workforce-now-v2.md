# Perform a Baseline Extraction of Payroll Interface for ADP Workforce Now V2

`⚙️ Steps to Enable`

> 📖 [View on Oracle Help Center](https://docs.oracle.com/en/cloud/saas/readiness/hcm/26c/hure-26c/26C-hr-wn-f49858.htm)

Perform a baseline extraction for Payroll Interface for ADP Workforce Now V2 to ensure that a full file doesn't get extracted after uptaking this update.

Route changes have been delivered to to report the state tax code together with the SOC code. That is, whenever an SOC code is reported, the worked state tax code will be reported along with it.

**NOTE**: You should perform a baseline extraction after applying this patch to avoid a full file extraction.

Perform regular data changes and extractions after the baseline extraction.

This ensures that a full file doesn't get extracted in the Payroll Interface for ADP Workforce Now V2.

## ⚙️ Steps to Enable and Configure

You don't need to do anything to enable this feature.
