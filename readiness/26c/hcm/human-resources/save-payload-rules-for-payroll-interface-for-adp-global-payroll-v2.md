# Save Payload Rules for Payroll Interface for ADP Global Payroll V2

`⚙️ Steps to Enable`

> 📖 [View on Oracle Help Center](https://docs.oracle.com/en/cloud/saas/readiness/hcm/26c/hure-26c/26C-hr-wn-f50109.htm)

Save the payload rules you are using (without making any changes) to integrate with ADP Global Payroll to uptake the changes delivered in the underlying attributes. Alternatively, a new payload rule can also be created to uptake the changes delivered in the underlying attributes.

Changes have been delivered to include the following additional attributes in SalaryDetails:

AnnualFullTimeSalary

  AnnualFTESalaryValue

  AnnualizationFactor

  SourceAmount

Performing a save ensures the changes delivered to the attributes are available in the payload rules.

## ⚙️ Steps to Enable and Configure

You don't need to do anything to enable this feature.
