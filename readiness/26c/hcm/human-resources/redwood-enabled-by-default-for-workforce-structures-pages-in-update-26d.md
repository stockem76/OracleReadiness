# Redwood Enabled by Default for Workforce Structures Pages in Update 26D

`⚙️ Steps to Enable` · `💡 Tips & Considerations`

> 📖 [View on Oracle Help Center](https://docs.oracle.com/en/cloud/saas/readiness/hcm/26c/hure-26c/26C-hr-wn-f49853.htm)

Enhance the user experience by letting users directly access the following Redwood-enabled Workforce Structures pages where the Redwood profile option is turned on by default in 26D.

This table shows the Redwood workforce structure pages where the profile options will be enabled by default from update **26D**onward:

Redwood Page | Profile Option Code | First Release of Redwood Page
Jobs | ORA_PER_JOBS_REDWOOD_ENABLED | 23B
Locations | ORA_PER_LOCATIONS_REDWOOD_ENABLED | 23B
Departments | ORA_PER_DEPARTMENTS_REDWOOD_ENABLED | 23B
Enterprise HCM Information | ORA_PER_ENTERPRISE_HCM_INFORMATION_REDWOOD_ENABLED | 24C
Grade Ladders | ORA_PER_GRADE_LADDERS_REDWOOD_ENABLED | 25A

Users can quickly and easily access the Global Human Resources (GHR) Redwood pages by using the quick action for the respective page.

## ⚙️ Steps to Enable and Configure

You don't need to do anything to enable this feature.

## 💡 Tips and Considerations

If you want to, you can disable the profile option (set to **No**) for the Redwood pages where it's enabled. To do so, follow these steps:

1. Navigate to the Setup and Maintenance work area.
2. Search for and click the **Manage Administrator Profile Values** task.
3. Search for the profile option code that you want to disable and select the profile option in the search results.
4. In the Profile Values area, enter **No**in the **Profile Value** field.
5. Click **Save and Close**.
6. Click **Done**.
