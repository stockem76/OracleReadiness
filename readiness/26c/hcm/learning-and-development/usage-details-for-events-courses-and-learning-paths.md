# Usage details for events, courses, and learning paths

`⚙️ Steps to Enable`

> 📖 [View on Oracle Help Center](https://docs.oracle.com/en/cloud/saas/readiness/hcm/26c/lear-26c/26C-learning-wn-f49566.htm)

The Learning Catalog page now makes it easier to review relationships across events, courses, and specializations. The **Where Used** columns are clickable, so you can open related entities directly. Related learning also appears in organized sections that match the details page layout.

Where Used Drawer with Clickable Links

Improve visibility into relationships and dependencies across catalog learning. Support more consistent catalog management and dependency oversight across learning structures.
