# View Real-Time Skill Gap Recommendations for Favorite Careers

`⚙️ Steps to Enable`

> 📖 [View on Oracle Help Center](https://docs.oracle.com/en/cloud/saas/readiness/hcm/26c/lear-26c/26C-learning-wn-f49343.htm)

When an employee adds a career role to their favorites in Opportunity Marketplace, the **Skills and qualifications for the {Career Role Name} role**swim lanes appear in real-time in Grow for that career role. The gap recommender process for the career of interest skill gaps runs immediately, facilitating the immediate availability of learning recommendations.

Similarly, when an employee removes a career role from favorites, the swim lanes disappear from Grow in real-time.

The prerequisite for this feature is that the employee should be included in the recommendation profile for generating the career of interest skill gaps.

**Business Benefit:** This feature helps employees take immediate action towards closing their skill gaps, plan their learning path more efficiently, and reduce the waiting time for accessing relevant career development information.

## ⚙️ Steps to Enable and Configure

You don't need to do anything to enable this feature.

## 📚 Key Resources

For information on running the gap recommender process, seethe topic,Set Up Swim Lane Data Using Recommendation Profiles, in Oracle Help Center.
