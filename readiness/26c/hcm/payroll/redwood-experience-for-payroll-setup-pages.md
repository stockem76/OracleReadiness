# Redwood Experience for Payroll Setup Pages

`⚙️ Steps to Enable` · `💡 Tips & Considerations` · `🔐 Access Requirements`

> 📖 [View on Oracle Help Center](https://docs.oracle.com/en/cloud/saas/readiness/hcm/26c/payr-26c/26C-payroll-wn-f49372.htm)

The following Payroll setup pages are now available in the Redwood user experience:

• Element Classifications
• Consolidation Group
• Payment Methods Preferences

Oracle Search for HCM is now integrated within these setup pages.

Get the most out of the new Redwood user experience with its modern design and enhanced usability.

## ⚙️ Steps to Enable and Configure

You don't need to do anything to enable this feature.

## 💡 Tips and Considerations

• All required profile options are enabled by default: 
  • Element Classification: ORA_PAY_PAYROLL_ELEMENTS_REDWOOD_ENABLED
• Consolidation Group: ORA_PAY_PAYROLL_DEFINITIONS_REDWOOD_ENABLED
• Payment Methods Preferences: ORA_PAY_PAYROLL_PAYMENTS_REDWOOD_ENABLED
• Central profile option **Enable VBCS Progressive Web Application User Interfaces across HCM application**: ORA_HCM_VBCS_PWA_ENABLED
• If you've already personalized any of the existing pages, you need to personalize them again in the Redwood user experience.

## 🔐 Access Requirements

Access to these new Redwood setup pages is controlled using the same security privileges that control access to the responsive setup pages. For information on existing security privileges, see Security Reference for HCM.
