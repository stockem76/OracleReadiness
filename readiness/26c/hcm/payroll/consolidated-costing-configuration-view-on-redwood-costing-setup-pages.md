# Consolidated Costing Configuration View on Redwood Costing Setup Pages

`⚙️ Steps to Enable`

> 📖 [View on Oracle Help Center](https://docs.oracle.com/en/cloud/saas/readiness/hcm/26c/payr-26c/26C-payroll-wn-f43502.htm)

You can now view costing segment enablement details directly on the Redwood costing setup pages. This enhancement shows which costing segments are enabled at each costing level, helping you quickly determine where specific segments are available for data entry. Where applicable, the pages also display debit and credit settings for element classifications.

Each costing page shows only the segment details relevant to that costing level. For example:

• Position costing displays only costing account segment information.
• Department costing displays default account, suspense account, and cost account segment information.

Select **More Actions** (three dots) **> View Costing Configuration** on the edit page to view this information or click **View Costing Configuration** when creating a new entry. In the Create flow, select a **Legislative Data Group** (LDG) to enable the button, because costing configuration is LDG-specific.

Gain greater visibility and control over costing setups by easily identifying which segments are enabled at each level. This streamlines configuration, reduces errors in data entry, and ensures compliance with your organization’s accounting requirements.

## ⚙️ Steps to Enable and Configure

You don't need to do anything to enable this feature.
