# Review Tax Compliance Updates

`⚙️ Steps to Enable` · `🔐 Access Requirements`

> 📖 [View on Oracle Help Center](https://docs.oracle.com/en/cloud/saas/readiness/hcm/26c/payr-26c/26C-payroll-wn-f50282.htm)

We have made it easier for you to review the changes made to tax info used by the US Oracle Tax Engine (USOPTE) during payroll processing. The Compliance Updates feature helps you review the changes delivered through the Legislative Data Updates process, including:

• Number of employees affected
• Where they are
• Effective dates of the changes

Sample Compliance Updates Page

This feature enables payroll administrators review legislative updates and assess their impact on employee tax calculations.

## ⚙️ Steps to Enable and Configure

You don't need to do anything to enable this feature.

## 🔐 Access Requirements

You must have the PAY_VIEW_PAYROLL_COMPLIANCE_UPDATES_PRIV functional security and privileges to view Compliance Updates
