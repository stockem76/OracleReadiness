# Real Time Information - Encryption for Security Compliance

`⚙️ Steps to Enable`

> 📖 [View on Oracle Help Center](https://docs.oracle.com/en/cloud/saas/readiness/hcm/26c/payr-26c/26C-payroll-wn-f49513.htm)

The latest encryption method has been adopted to protect RTI authentication details. Oracle supports both existing and enhanced encryption methods, enabling customers to use the latest standards for secure RTI data processing.

Enhances data protection through the adoption of the latest encryption methods for RTI processing.

## ⚙️ Steps to Enable and Configure

You need to run the feature upgrade process to update all existing records. To do this:

1. Go to  **My Client Groups****-> Payroll ->****Submit a Flow.**
2. Select the appropriate LDG.
3. Search for the **Run Feature Upgrade**flow.
4. Enter a name for the payroll flow.
5. Search for and select **RTI Upgrade for UK Employers**in the required parameters field
6. Submit the flow.
