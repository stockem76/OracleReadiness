# Time Change Request enhancements

`⚙️ Steps to Enable` · `💡 Tips & Considerations`

> 📖 [View on Oracle Help Center](https://docs.oracle.com/en/cloud/saas/readiness/hcm/26c/tila-26c/26C-time-labor-wn-f49901.htm)

Time change requests now include request-level Comments that apply to the request as a whole. Time entry comments remain separate and are labeled Time Entry Comment.

Request comments appear in the approval notification and approval UI.

Employee Time Change Requests and Team Change Requests include clearer Request Date and Submission Date labels and filters.

Allows employees to provide reasons for change requests as comments while providing more context for managers and timekeepers for their approval decisions. Enables easier search using filters on the change request dates.

## ⚙️ Steps to Enable and Configure

None

## 💡 Tips and Considerations

The 'Comments required on Request Time Changes' extensibility constant applies to the request-level Comments field. The default setting is Not Required.
