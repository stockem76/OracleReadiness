# Redwood Experience: Enhanced Editing for Private Hiring Events

`⚙️ Steps to Enable` · `💡 Tips & Considerations` · `🔐 Access Requirements`

> 📖 [View on Oracle Help Center](https://docs.oracle.com/en/cloud/saas/readiness/hcm/26c/recr-26c/26C-recruiting-wn-f44533.htm)

Recruiters and event owners can now edit the date, time, and location (physical address or virtual meeting link) for private hiring events, as long as an event start date and time are still in the future, and the event is still open for registration. Once the details are changed and saved, all registered attendees are notified of the changes.

**Business benefit:**This enhancement reduces event cancellations and data loss by enabling edits to private hiring events during registration. It improves the candidate experience with timely notifications of event changes.

## ⚙️ Steps to Enable and Configure

For privileges, see the **Access Requirements** section.

## 💡 Tips and Considerations

• When reposting private events, previously associated career sites are preselected and can't be removed but new sites can be added.
• Private events can be scheduled for posting in advance, aligning posting options with public event workflows.
• Some edits require moving the private event from open for registration to an unposted state before changes can be made

## 🔐 Access Requirements

Users must have the event owner or recruiter role to edit private hiring event details.
