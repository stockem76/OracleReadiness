# Workflow Agent to Power AI Assist in External Candidate Experience

`⚙️ Steps to Enable` · `💡 Tips & Considerations`

> 📖 [View on Oracle Help Center](https://docs.oracle.com/en/cloud/saas/readiness/hcm/26c/recr-26c/26C-recruiting-wn-f44462.htm)

Use a workflow agent to power AI Assist features in career site pages. AI worker agents are designed to be reusable and channel agnostic for broader extensibility. This replaces deprecated Prompt Lab prompts while preserving the same content-generation and regeneration experience for supported fields. These agents are included by default and are customizable through AI Agent Studio:

• Generating intelligent asset element in job details page. Agent Team Code: ORA_CAREER_SITE_KEYWORD_BASED_ASSET_SELECTION
• Job fit metrics element in the job details page. Agent Team Code: ORA_CAREER_SITE_JOB_MATCH_SNIPPET
• Introduction text generation to use on the category landing page. Agent Team Code: ORA_CAREER_SITE_JOB_CATEGORY_PAGES
• Content generation for paragraph element while creating career site pages. Agent Team Code: ORA_CAREER_SITE_CONTENT_ASSISTANT
• Generating job summary to use in smart search and Career Coach job recommendations. Agent Team Code: ORA_CAREER_SITE_JOB_SUMMARY_FOR_SEARCH

**Business benefit:** This enhancement modernizes Oracle Recruiting AI agent architecture and aligns with Oracle’s centralized workflow framework.

## ⚙️ Steps to Enable and Configure

There's no setup for this feature. For additional details, see Deprecation of AI Configurator and Transition to AI Agent Studio

## 💡 Tips and Considerations

The workflow model is the strategic architecture moving forward. Future enhancements and bug fixes will only be supported in this model.
