# Add Pool Status in Candidate Pool Reports

`⚙️ Steps to Enable` · `💡 Tips & Considerations`

> 📖 [View on Oracle Help Center](https://docs.oracle.com/en/cloud/saas/readiness/hcm/26c/recr-26c/26C-recruiting-wn-f44279.htm)

You can add Pool Status and Pool Status Code in your candidate pool reports to know which pools are active or inactive. These new fields are available in:

• Recruiting – Recruiting Real Time > Candidate Pool > Candidate Pool – Basic Information

**Business benefit:** Users have a way to identify active and inactive pools which enhances reporting on pools.

## ⚙️ Steps to Enable and Configure

You don't need to do anything to enable this feature.

## 💡 Tips and Considerations

Including the Pool Status and filtering on it will better serve users having access to all candidate pools, which is possible by creating a Report All Candidate Pools. For details, see Provide Access to All Pools.
