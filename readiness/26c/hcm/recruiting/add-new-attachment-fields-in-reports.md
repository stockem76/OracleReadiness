# Add New Attachment Fields in Reports

`⚙️ Steps to Enable`

> 📖 [View on Oracle Help Center](https://docs.oracle.com/en/cloud/saas/readiness/hcm/26c/recr-26c/26C-recruiting-wn-f44282.htm)

You can add these new attachment fields in your reports:

• Category Active Flag
• Category Active Flag Code
• Category Verification Required Flag
• Category Verification Required Flag Code
• Attachment Status Code

These new fields are available in:

• Recruiting – Recruiting Real Time > Job Application > Job Application - Attachments
• Recruiting – Hiring Events Real Time > Job Application > Job Application - Attachments

**Business benefit:**When listing all attachments associated to candidates and applications in your reports, you can know if the attachments are approved or if candidates have updated their attachments, as requested, without having to open each candidate record in the application.

## ⚙️ Steps to Enable and Configure

You don't need to do anything to enable this feature.
