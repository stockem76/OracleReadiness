# Add Country Code and Area Code in Reports

`⚙️ Steps to Enable`

> 📖 [View on Oracle Help Center](https://docs.oracle.com/en/cloud/saas/readiness/hcm/26c/recr-26c/26C-recruiting-wn-f44280.htm)

Use the country code and area code to obtain complete primary phone numbers in your recruiting reports for all user types and from all subject areas.

Here’s the list of new entries in the Recruiting - Recruiting Real Time subject area. You'll find similar entries in all Recruiting subject areas.

• Candidate > Candidate Interaction    
• Added By Person Phone Country Code
• Added By Person Phone Area Code
• Candidate Pool  >  Pool Owners    
• Owner Phone Country Code
• Owner Phone Area Code
• Candidate Pool  > Pool Member Candidates / Pool Member Candidates - Interaction    
• Added By Person Phone Country Code
• Added By Person Phone Area Code
• Interview Scheduling  >  interview Participant    
• Interview Participant Phone Country Code
• interview Participant Phone Area Code
• Job Application  > Job Application - Interaction Note    
• Added By Person Phone Country Code
• Added By Person Phone Area Code
• Job Offer  > Job Offer - Offer Team
• Collaborator Phone Country Code
• Collaborator Phone Area Code
• Hiring Manager Phone Area Code
• Hiring Manager Phone Country Code
• Recruiter Phone Area  Code
• Recruiter Phone Country Code
• Job Requisition  > Job Requisition - Hiring Team    
• Collaborator Phone Country Code
• Collaborator Phone Area Code
• Hiring Manager Phone Area Code
• Hiring Manager Phone Country Code
• Recruiter Area Phone Code
• Recruiter Phone Country Code
• Source Tracking  > Source Tracking - Added By Person    
• Added By Person Phone Country Code
• Added By Person Phone Area Code

**Business benefit:** Easily call users directly from the reports, without having to verify if the user is local with same area code.

## ⚙️ Steps to Enable and Configure

You don't need to do anything to enable this feature.
