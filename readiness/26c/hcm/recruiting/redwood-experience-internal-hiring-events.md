# Redwood Experience: Internal Hiring Events

`⚙️ Steps to Enable` · `🔐 Access Requirements`

> 📖 [View on Oracle Help Center](https://docs.oracle.com/en/cloud/saas/readiness/hcm/26c/recr-26c/26C-recruiting-wn-f44532.htm)

Organizations can now create and manage internal hiring events to promote open roles to existing employees through Oracle Recruiting Hiring Events. Internal events are posted to Opportunity Marketplace.

The feature includes:

• Support for multiple audience types: internal, internal and external, and external
• Internal employee registration capabilities.
• Ability to change the event theme.

**Business benefit:** This feature lets recruiters post and manage events targeted specifically for internal employees, streamlining internal mobility initiatives.

## ⚙️ Steps to Enable and Configure

1. In the Setup and Maintenance work area, enable Oracle Recruiting Hiring Events.
2. Configure recruiter security privileges. See the **Access Requirements** section.
3. When creating a hiring event, select one of the following audience types: Internal, External, or Internal and External.

Select Internal for the Event Type
