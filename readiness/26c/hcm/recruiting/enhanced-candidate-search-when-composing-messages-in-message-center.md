# Enhanced Candidate Search When Composing Messages in Message Center

`⚙️ Steps to Enable`

> 📖 [View on Oracle Help Center](https://docs.oracle.com/en/cloud/saas/readiness/hcm/26c/recr-26c/26C-recruiting-wn-f46389.htm)

When composing a message in Message Center, you can now search for candidates based on their name or email address.

You can enter partial or full words in the candidate search list.

Candidate Search Based on Email Address

This enhancement applies only to the Compose action in Message Center and not to the Search box in the message list. The functionality of the Search box remains unchanged.

**Business Benefit:** This enhancement facilitates faster search when sending candidate communications.

## ⚙️ Steps to Enable and Configure

You don't need to do anything to enable this feature.

## 📚 Key Resources

For information on the Search box in the message list view, see Search for Messages in Oracle Help Center.
