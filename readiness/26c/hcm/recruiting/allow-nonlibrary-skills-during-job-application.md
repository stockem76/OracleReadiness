# Allow Nonlibrary Skills During Job Application

`⚙️ Steps to Enable`

> 📖 [View on Oracle Help Center](https://docs.oracle.com/en/cloud/saas/readiness/hcm/26c/recr-26c/26C-recruiting-wn-f44499.htm)

When Skills Center is enabled, you can control whether external candidates who apply for jobs can add custom non-library skills in their job applications.

**Business benefit:** This enhancement help you prevent the capture of inappropriate data.

## ⚙️ Steps to Enable and Configure

Select the **Allow adding nonlibrary skills** checkbox for **Skills center**to indicate whether external candidates can add custom non-library skills when they apply for jobs. For details, see How do I Allow Nonlibrary Skills in Skills Center and Team Skills Center?
