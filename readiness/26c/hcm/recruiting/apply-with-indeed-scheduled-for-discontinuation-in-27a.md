# Apply with Indeed Scheduled for Discontinuation in 27A

`⚙️ Steps to Enable`

> 📖 [View on Oracle Help Center](https://docs.oracle.com/en/cloud/saas/readiness/hcm/26c/recr-26c/26C-recruiting-wn-f46330.htm)

Starting in release 27A, the Apply with Indeed feature will be phased out, which means candidates won't be able to use their Indeed profile while applying on Oracle career sites. This change is driven by Indeed’s update to their integration model, which includes the retirement of the APIs currently used by the existing Apply with Indeed functionality.

Oracle Recruiting introduced Direct Apply with Indeed in the 25D release. This newer integration enables candidates to apply using their Indeed profile through an updated mechanism directly from Indeed platform.

Moving forward, customers are encouraged to adopt Direct Apply with Indeed to ensure uninterrupted functionality and to take advantage of the new integration.

**Business benefit:** Since its introduction, Direct Apply with Indeed has proven to be highly successful, offering a smoother application experience for candidates.
