# Redwood Experience: Express Communication Template for Employee Event Invitations

`⚙️ Steps to Enable` · `🔐 Access Requirements`

> 📖 [View on Oracle Help Center](https://docs.oracle.com/en/cloud/saas/readiness/hcm/26c/meex-26c/26C-employee-experience-wn-f44531.htm)

Use the Express Communication templates to send invitations to employee event teams. This template lets you quickly select a preconfigured list of people who will work on an event, and simplifies event promotion creation by automatically including a sender profile, HCM list and eligibility profile configured for the team, and predefined email template/content. This bundled setup ensures faster and more consistent communication configuration.

**Business benefit:**This feature lets event owners efficiently send standardized event notifications to employees. without having to manually configure multiple components.

## ⚙️ Steps to Enable and Configure

1. Set up an Express Email Template and Communication Team. For details see Express Communication Teams.
2. While creating an event, select an Express Communication Template.

Select an Express Communication Template

For privileges, see the **Access Requirements** section.

## 🔐 Access Requirements

To see the Manage Events tab, users need this privilege: IRC_MANAGE_EMPLOYEE_EVENT_PRIV.
