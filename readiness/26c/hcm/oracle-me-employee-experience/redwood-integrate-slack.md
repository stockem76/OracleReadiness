# Redwood: Integrate Slack

`⚙️ Steps to Enable`

> 📖 [View on Oracle Help Center](https://docs.oracle.com/en/cloud/saas/readiness/hcm/26c/meex-26c/26C-employee-experience-wn-f46312.htm)

Set up Oracle Communicate to integrate with Slack to send campaigns as messages to employees. To start the configuration, you need a Slack workspace admin to carry out these tasks:

• Create and register the Slack app and bot
• Configure the required OAuth scopes and event subscriptions
• Install the app to approved workspaces

Then the campaign admin integrates the Slack app with Oracle Communicate and determines which Slack workspaces and channels (public and approved private) the bot can access.

After that, the campaign manager can carry out these tasks:

• Select Slack as a communication channel
• Compose messages using a rich text editor using AI assistance
• Schedule campaigns
• Review a summary with AI support of reactions and responses

Lastly, the employee can interact with the Slack message through emoji reactions and thread replies.

**Business benefit**: Campaign managers can now post and collaborate with Slack to increase employee engagement and communication.
