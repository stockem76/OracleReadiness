# Reporting on Celebrate Posted Recognitions Columns

`⚙️ Steps to Enable` · `💡 Tips & Considerations`

> 📖 [View on Oracle Help Center](https://docs.oracle.com/en/cloud/saas/readiness/hcm/26c/meex-26c/26C-employee-experience-wn-f49861.htm)

Now you can report on new posted celebrations attributes relevant to the Celebration Fact Details in the Celebrate - Celebration Real Time subject area.

• This feature is functionally relevant only for Celebration Fact and not for Feed Summary Fact.
• A new folder called "Posted Celebrations" has been added under the "Award – Details" folder within "Celebrate Fact Details".

Posted Celebrations

Extend your ability to report on data in the Celebrate - Celebration Real Time subject area. The new "Posted Celebrations" folder provides additional details to indicate whether the award has been posted or not, helping users track award status.

## ⚙️ Steps to Enable and Configure

You don't need to do anything to enable this feature.

## 💡 Tips and Considerations

• Element entries details are already captured in the subject area for a given program under the folder Celebration Program Elements.
• A new folder "Celebrations" has been introduced under the existing folder "Feed Summary Fact Details", as a copy of the existing "Celebrations" folder. There's no regression on existing reports.
• The "Celebration Visibility" folder has been moved to the parent level to enable the "Visibility" dimension to work with "Fact - Celebrate - Recognition Feed Summary" metrics. Previously, it was a child of the "Celebrate Fact Details" folder.
