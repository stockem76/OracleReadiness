# Redwood: New Administrator Skills Center Experience

`⚙️ Steps to Enable` · `💡 Tips & Considerations`

> 📖 [View on Oracle Help Center](https://docs.oracle.com/en/cloud/saas/readiness/hcm/26c/dyns-26c/26C-dynamic-skills-wn-f49670.htm)

Displays skills related to an employee for skills administrator, in new Skills Center (*Talent Profile Search > Talent Profile Details > Skill Center Section “Visit Skills Center”*), by grouping them in tabs as follows:

• Required – Skills assigned by a manager
• Self Developing – Skills voluntarily assigned by the employee
• All Skills

Skills administrator dashboard for viewing the skills of an employee when rating model is enabled

The primary difference between a manager’s and skills administrator’s dashboard is that an administrator can’t edit any details of an employee and can only view them.

Provides skills administrator with a holistic, actionable view of any employee's skills. With this enhanced experience, skills administrator can:

• Quickly see required, self-developing skills, and all skills associated with each employee, improving visibility and assessment of skill gaps and development areas.
• View detailed skill information for each skill, including the skill chip, the assignment source type, validation level, employee’s self-rating, rating guidance, and manager rating.

## ⚙️ Steps to Enable and Configure

You have enabled:

• Enhanced Dynamic Skills. For details, see Pathways to Enhanced Dynamic Skills.
• Enhanced Skills Center. For details, see **How can i enable Enhanced Skills Center?**

Rating levels can be displayed as circular or textual. Circular ratings are supported only when there are 7 or fewer rating levels. If there are more than 7 rating levels, the display switches to text-based ratings. If the display is set to textual, rating levels are displayed as text regardless of the number of rating levels. Select how the rating levels are displayed. For details, see **How can I enable circle rating?**

Define the order and visibility of tabs. For details, see How can I control the visibility and order of columns in Enhanced Skills Center?

Select a default tab, see How can I set a default tab in Enhanced Skills Center?

## 💡 Tips and Considerations

• On the Skills Settings page, the option to allow any employee to add skills to another employee’s profile is not valid for Enhanced Skills Center.
