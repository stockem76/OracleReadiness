# AI Agent: Intelligent Talent Profile Search for Team Skills Center

`⚙️ Steps to Enable` · `🔐 Access Requirements`

> 📖 [View on Oracle Help Center](https://docs.oracle.com/en/cloud/saas/readiness/hcm/26c/dyns-26c/26C-dynamic-skills-wn-f49675.htm)

Delivers Intelligent Talent Profile Search , which is a chat-based talent-finding agent for managers. Users ask natural-language questions about people profiles, and the agent returns ranked employees, usually as an HTML table, with names linked to each person’s talent profile.

## 🎯 Business Benefit

Assists HR specialists in finding an ideal candidate who matches the specified criteria by combining talent profiles and assignment details with flexible queries.

## ⚙️ Steps to Enable and Configure

Enable Intelligent Talent Profile Search for Team Skills Center. For details, see How can I enable Intelligent Talent Profile Search for Talent Profile or Team Skills Center?

## 🔐 Access Requirements

This feature uses Intelligent Talent Profile Search agent. The following Spectra Authorization Service (SAS) duty role is required to access the agent:

• Intelligent Talent Profile Search Duty Role

This is added by default to the delivered Line Manager role. If you're creating a custom role, ensure that the duty role is added to it.

The agents you can view depend on the roles and privileges assigned to you. To access the Intelligent Talent Profile Search agent, your role must be explicitly granted access to it. See How can I give users access to AI agents and Access Requirements for AI Agent Studio.
