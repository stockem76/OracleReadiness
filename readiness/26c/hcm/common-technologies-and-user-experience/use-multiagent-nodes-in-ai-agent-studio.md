# Use Multiagent Nodes in AI Agent Studio

`⚙️ Steps to Enable` · `🔐 Access Requirements`

> 📖 [View on Oracle Help Center](https://docs.oracle.com/en/cloud/saas/readiness/common/26c/common26c/26C-common-wn-f50327.htm)

You can now use Multiagent nodes to build agentic flows that use a supervisor agent to coordinate multiple specialized worker agents within a workflow. The supervisor agent analyzes user requests, determines intent, and dynamically routes tasks to the most appropriate child agent based on its role and expertise. Each worker agent can be configured with its own responsibilities, prompts, and behavior. This process enables the supervisor agent to orchestrate tasks without requiring manually defined routing paths.

This feature enables organizations to build scalable multiagent experiences by distributing work across specialized worker agents. Dynamic agent orchestration simplifies workflow design, reduces development effort, and helps deliver accurate, contextually relevant responses to a wide range of user requests.
