# Oracle ADF Desktop Integration Add-In

`⚙️ Steps to Enable`

> 📖 [View on Oracle Help Center](https://docs.oracle.com/en/cloud/saas/readiness/common/26c/common26c/26C-common-wn-f47430.htm)

No changes. In release 26C, version 5.2.5.29987 of the Oracle ADF Desktop Integration add-in is being used, which is the same as in the last release.

You can download the Oracle ADF Desktop Integration add in version 5.2.5.29987, if not done already. For more information about this version, see Updated Oracle ADF Desktop Integration Add-In (26B).

You need to install the add-in on each desktop. For more information, see How to install the ADF Integration Add-in for Excel (KB105801) on My Oracle Support.

## ⚙️ Steps to Enable and Configure

You don't need to do anything to enable this feature.
