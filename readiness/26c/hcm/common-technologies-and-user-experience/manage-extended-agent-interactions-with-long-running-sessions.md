# Manage Extended Agent Interactions with Long-Running Sessions

`⚙️ Steps to Enable` · `🔐 Access Requirements`

> 📖 [View on Oracle Help Center](https://docs.oracle.com/en/cloud/saas/readiness/common/26c/common26c/26C-common-wn-f47426.htm)

You can now manage extended, including multi-day, interactions with agents. If the system needs time to respond to a complex query, or if you need to leave a conversation to gather data, switch to another conversation, or take a break, you can rejoin the same conversation in progress.

This capability ensures that you experience continuity, traceability, and control over your interactions.

## ⚙️ Steps to Enable and Configure

You must have access to use AI Agent Studio.
