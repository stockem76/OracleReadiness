# New look and feel, and features for AI Agent Studio

`⚙️ Steps to Enable` · `🔐 Access Requirements`

> 📖 [View on Oracle Help Center](https://docs.oracle.com/en/cloud/saas/readiness/common/26c/common26c/26C-common-wn-f49883.htm)

AI Agent Studio is moving to a more intelligent user experience for building agents and agentic applications. Here are the new features available:

• New, modern visual design
• Simplified navigation controls
• Visual builders for agentic applications and workflows
• Builder assistant providing natural language assistance with building apps and agents, prompting, and debugging
• Agent debugging tools
• Additional document processing capabilities
• Deterministic policy models
• Custom approval processing
• Extensive library of connectors
• AI Agent Studio Command Line Interface (CLI)

AI Agent Studio Home Page

Some of the features such as **Marketplace**and **Monitoring and Evaluation** will render in the former user experience.

You can accelerate the development of your agents and agentic apps with the new advanced building and debugging capabilities, and you can also expand your agent capabilities using the new features available.
