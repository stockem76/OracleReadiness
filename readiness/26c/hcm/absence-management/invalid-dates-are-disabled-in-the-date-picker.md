# Invalid Dates are Disabled in the Date Picker

`⚙️ Steps to Enable`

> 📖 [View on Oracle Help Center](https://docs.oracle.com/en/cloud/saas/readiness/hcm/26c/amg-26c/26C-absence-mgmt-wn-f49555.htm)

When you enter absences against past or future assignments using the enhanced Redwood pages, the date picker now allows selection of only the valid dates for the chosen assignment. The invalid dates are greyed out. This applies to only the absence types that allow assignment selection and when an assignment is selected.

Improve accuracy when recording absences by guiding users to select only valid dates for an assignment. This reduces errors and rework, delivering a smoother, smarter absence entry experience in Redwood.

## ⚙️ Steps to Enable and Configure

You don't need to do anything to enable this feature.
